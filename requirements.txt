# README.md
# Tibia Multi-Character Detector

System monitorujący graczy online w Tibia i wykrywający potencjalne multi-chary.

## Instalacja lokalna

1. Sklonuj repozytorium
2. Zainstaluj zależności:
   ```bash
   pip install -r requirements.txt
   ```
3. Uruchom aplikację:
   ```bash
   python app.py
   ```

## Deployment na Railway

1. Stwórz konto na Railway.app
2. Połącz z GitHub
3. Deploy automatycznie się rozpocznie

## Jak to działa

- **Scraping**: Co 10 minut pobiera listę graczy online
- **Analiza**: Porównuje wzorce logowania
- **Wykrywanie**: Znajduje pary graczy, które nigdy nie są online razem
- **Scoring**: Oblicza prawdopodobieństwo multi-chara (0-100%)

## Algorytm

System wykrywa multi-chary na podstawie:
1. Brak jednoczesnego bycia online
2. Szybkie zmiany między postaciami (< 10 min)
3. Podobne wzorce aktywności
4. Regularne naprzemienne logowania
